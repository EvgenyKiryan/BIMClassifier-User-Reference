BIM Classifier
==============

**BIM Classifier** — это расширение для САПР `Renga <https://rengabim.com/>`_, созданное для решения задач по классификации объектов **цифровой информационной модели (ЦИМ)**.

Одной из главной особенностей расширения является возможность построения собственной классификационной иерархии объектов ЦИМ по правилам пользователя. Правила указываются в файле формата `JSON <https://www.json.org/json-ru.html>`_ и состоят из следующих JSON-объектов:
